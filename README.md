# C++ Foundations for Low-Latency Systems

A from-scratch beginner project that teaches the C++ a low-latency trading engine rests on — structs and value semantics, references vs copies, std::vector and enum class, and the const-correctness and ownership habits that keep hot-path code fast. The anchor is a tiny Order/Trade model you grow into a real engine later in the journey.


## Stack
- C++
- value semantics
- references
- std::vector
- enum class
